# Развертывание на Vercel

## Подготовка

1. Установите Vercel CLI:
```bash
npm i -g vercel
```

2. Авторизуйтесь в Vercel:
```bash
vercel login
```

## Развертывание

1. Инициализируйте проект в Vercel:
```bash
vercel
```

2. Следуйте инструкциям:
   - Выберите "Y" для создания нового проекта
   - Выберите текущую директорию
   - Настройте параметры проекта

3. Настройте переменные окружения в Vercel:
   - Перейдите в dashboard Vercel
   - Откройте ваш проект
   - Перейдите в Settings > Environment Variables
   - Добавьте переменную:
     - Name: `TELEGRAM_BOT_TOKEN`
     - Value: `8593913204:AAFRpJqdRcHhknFYnKdUK_Lt3vV5smO2zcc`

4. Запустите развертывание:
```bash
vercel --prod
```

## Структура проекта

- `server/index.ts` - Основной Express сервер
- `client/` - React клиентское приложение
- `shared/` - Общие типы и схемы
- `vercel.json` - Конфигурация Vercel

## Особенности развертывания

- API маршруты автоматически направляются на сервер
- Статические файлы обслуживаются из `client/dist`
- Используется SQLite база данных (файл `sqlite.db`)

## Переменные окружения

- `TELEGRAM_BOT_TOKEN` - Токен Telegram бота
- `NODE_ENV` - Окружение (автоматически устанавливается Vercel)

## После развертывания

1. Обновите URL в коде Telegram бота (в `server/bot.ts`)
2. Проверьте работу всех функций
